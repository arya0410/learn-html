HTML Basics

Learn the fundamentals of HTML.

HTML, Head, Title, & Body Tags

  • HTML documents are built with <html>, <head>, <title>, and <body> tags.
  • <html> encloses the entire document, <head> contains metadata, <title> sets the page title, and <body> holds visible content.

Headings, Paragraphs, & Lists

  • Structural elements like <h1> to <h6> define headings with descending levels of importance.
  • Paragraphs are created using <p> tags.
  • Lists include unordered <ul> and ordered <ol> lists, with list items specified by <li>.

The Strong & Em Tags

  • <strong> is a semantic tag indicating strong importance or seriousness, often displayed as bold text.
  • <em> emphasizes text, typically rendering it in italics to convey stress or importance.

The Doctype

  • <!DOCTYPE html> declares the HTML version, ensuring proper rendering in browsers.
  • Crucial for compatibility and informing the browser how to interpret the document's code.

The Lang Attribute

  • The lang attribute in the <html> tag specifies the language of the document.
  • Aids accessibility tools and search engines in understanding the content's linguistic context.

The Meta Tag & the Unicode Character Set

  • <meta charset="UTF-8"> defines the character set for text encoding.
  • Supports a wide range of characters using Unicode, essential for multilingual content.

Coding Links: Absolute & Relative URLs

  • Links are created with the <a> tag, with the href attribute.
  • Absolute URLs point to external resources, while relative URLs are used for internal links.

Anchor Tags & Hrefs

  • <a> tags create hyperlinks.
  • The href attribute specifies the destination, whether it's a web page, an image, or another resource.

Linking to Other Websites

  • For linking to other websites, use absolute URLs in the href attribute.
  • This directs the browser to the exact location of the linked content on the web.

Linking to Pages Within a Website

  • Internal links within a website use relative URLs in the href attribute.
  • These URLs indicate the path to the target page or resource relative to the current location.

Opening a Link in a New Browser Window/Tab

  • To open a link in a new browser window or tab, use the target="_blank" attribute within the <a> tag.
  • This provides a convenient way to keep the current page open.

Adding Images

  • Images are added to a webpage using the <img> tag.
  • The src attribute specifies the image file's URL.

The Break Tag

  • The <br> tag is a self-closing tag that creates a line break within text or elements like paragraphs.

The Image Tag & Source Attribute

  • The <img> tag is used to embed images in HTML.
  • The src attribute defines the image file's source (URL).

Using Width, Height, & Alt Attributes

  • The <img> tag allows the specification of image attributes.
  • width and height define the dimensions, and alt provides alternative text for accessibility.

Using Horizontal Rules

  • The <hr> tag is a self-closing tag that inserts a horizontal line, creating visual separation between different sections of content on the webpage.

Intro to Cascading Style Sheets (CSS)

  • CSS enhances the presentation of HTML elements.
  • It allows for styling and layout customization, making web pages visually appealing and responsive to different devices.

The Style Tag

  • The <style> tag is used to embed CSS directly within an HTML document.
  • It allows for the definition of styles for various HTML elements, influencing their appearance and layout.

Tag Selectors

  • CSS employs tag selectors to target and apply styles to specific HTML elements.
  • For example, using <h1> as a selector will apply styles to all <h1> elements on the page.

The Font-Size, Font-Family, Color, & Line-Height Properties

  • CSS properties such as font-size, font-family, color, and line-height control the appearance of text.
  • font-size sets the size, font-family specifies the font type, color defines the text color, and line-height adjusts the space between lines.

Hexadecimal Color Codes

  • Hexadecimal color codes, like #ff0000 for red, are commonly used in CSS to define specific colors.
  • They provide a wide range of color options, allowing precise color selection for elements on a webpage.
